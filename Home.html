Siapa Yustinus?

Saya adalah Mahasiswa tahun ke-3 Jurusan Sistem Informasi di Universitas Bina Nusantara. Saya mendeskripsikan diri saya sebagai orang yang pekerja keras, teliti, jujur, dan bukan seorang deadliners. Jika saya punya waktu luang saat bekerja, saya memilih untuk mengerjakan tugas-tugas saya secepat mungkin tanpa menunda-nunda. Saya juga senang membuat Front-end pada suatu website. Website portofolio ini saya buat full coding dari nol tanpa menggunakan tools seperti Webflow, Mendix, Wordpress ataupun tools pembuatan website tanpa coding lainnya. Saya memiliki komputer (computer skill) di angka 6/10 mulai dari HTML, CSS, javascript, dan Figma. Saya juga memiliki pengetahuan mengenai cara desain Ui/Ux.

Pengalaman Organisasi

Saya mengikuti dua UKM (Unit Kerja Mahasiswa) di BINUS yaitu HIMSISFO (Himpunan Mahasiswa Sistem Informasi) sebagai anggota dan BNCC (Binusian Computer Club) sebagai anggota.
